Allmaps is a set of open source tools that make it easier and more fun to search, explore, georeference and work with collections of scanned maps.

Allmaps works with any image repository that supports IIIF.

All components rely on a simple JSON metadata structure that holds information about each map’s control points and mask. This metadata structure is part of an upcoming standard by the IIIF community and is based on W3C’s Web Annotation specification. For some examples of what this data looks like, see my IIIF Map Collections repository.

Allmaps and its components are far from finished. For now, you can have a look at the following things:

  • Allmaps viewer;
  • Allmaps editor;
  • GitHub;
  • Observable notebooks.
