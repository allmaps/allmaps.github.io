Allmaps is a set of open source tools that make it easier and more fun to find, explore and georeference map collections.

Allmaps works with any image repository that supports IIIF.

All components rely on a simple JSON metadata structure that holds information about each mapâ€™s control points and mask. This metadata structure is part of an upcoming standard by the IIIF community. For some examples of this metadata, see my IIIF Map Collections repository.
